# to-do-list
About Aplikasi "TO DO LIST" berbasis Java dibuat untuk membantu pengguna mencatat dan mengelola aktivitas harian berdasarkan hari dalam seminggu agar lebih terorganisir. Program ini memudahkan pengelolaan tugas, pemantauan penyelesaian aktivitas, meningkatkan produktivitas, serta mengurangi risiko lupa jadwal penting.
